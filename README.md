# Japan Tour — Luxury Travel Landing Experience

A premium Japan travel landing page crafted with modern web technologies to deliver an immersive and cinematic user experience.  
Built using **React**, **Vite**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

---

## Preview

A visually rich landing page inspired by luxury tourism in Japan, featuring:

- Smooth scroll interactions
- Cinematic motion design
- Responsive modern layouts
- Elegant typography
- Immersive visual storytelling

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/japantour.git
```

### Navigate to the Project

```bash
cd japantour
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

Application runs locally at:

```bash
http://localhost:5173
```

---

## Available Scripts

### Development

```bash
npm run dev
```

Starts the local development server.

### Production Build

```bash
npm run build
```

Creates an optimized production build inside the `dist` directory.

### Preview Production Build

```bash
npm run preview
```

Runs a local preview of the production build.
